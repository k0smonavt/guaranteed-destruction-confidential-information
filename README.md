# Приложение для гарантированного уничтожения конфиденциальной информациии

Приложение для гарантированного уничтожения  конфиденциальной информациии, которое использует консольный интерфейс. Оно предоставляет на выбор пользователя 3 способа уничтожения конфиденциальной информации: лгоритм Брюса Шнейера, американский национальный стандарт DoD 5220.22-M и собственный метод пользователя. 

# Принцип работы:

1. Пользователь сначала подтверждает, что хочет удалить информацию из файла, посредством написания слова «DeLeTe».
2. После чего вводит название файла с расширением.
3. Выбирает нужный ему метод:
   1) алгоритм Брюса Шнейера
   2) американский национальный стандарт DoD 5220.22-M
   3) собственный метод пользователя.
4. После получения первичных данных, приложение уведомляет пользователя об успешном выполнении программы.

Программа выполнена на языке программирования С, и умеет выполнять всё вышеперечисленное.
